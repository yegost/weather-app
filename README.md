# Weather App

A React weather app built with Vite and React that shows current weather and a 5-day forecast for any city.

## Features
- Search cities with autocomplete dropdown
- Current temperature, description, humidity, wind, feels like, sunrise and sunset
- 5-day forecast
- °C / °F toggle
- Dynamic background based on weather condition and temperature

## Tech
- React
- Vite
- OpenWeatherMap API
- CSS Modules
